# simple_user_manage_demo
SpringBoot+SpringSecurity+Thymeleaf整合实践，完成一个相对简单的用户信息管理系统。

### 技术选型
* SpringBoot
* Mybatis
* SpringSecurity
* BootStrap
* Thymeleaf
